# scales-python

[![PyPI](https://img.shields.io/pypi/v/scales-python)](https://pypi.org/project/scales-python/)

Python port of the R **scales** package.

## Installation

```bash
pip install scales-python
```

Or, for a local development checkout:

```bash
git clone https://github.com/Bio-Babel/scales-python.git
cd scales-python
pip install -e ".[dev]"
```

## Quick Start

```python
import scales
```

## Documentation

```bash
pip install -e ".[docs]"
mkdocs serve
```
